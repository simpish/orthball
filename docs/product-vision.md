## 1. コンセプト

**OrthBallは、3ユニット分離型のオーソリニアキーボードで、右手ユニットに55mmトラックボールを統合し、ホームポジションから手を離さずにすべてのPC操作を完結させるデバイスである。**

3Dプリントケースによる一体成型で、アクリル積層キーボード（On the 17等）の「高さ感」を継承しつつ、トラックボールを上面＋側面の両方から操作できる独自のエルゴノミクスを実現する。

## 2. 3ユニット構成 — 上面図

```
  ┌─────────────────┐   ┌───────────┐   ┌─────────────────┬───────┐
  │                 │   │           │   │                 │       │
  │   左手ユニット    │   │  中央テン  │   │   右手ユニット    │  ◉ TB │
  │   6×4 + thumb   │   │  キー 3×4  │   │   6×4 + thumb   │ Choc6 │
  │   MX + 曲面     │   │  MX flat  │   │   MX + 曲面     │       │
  │                 │   │           │   │                 │       │
  └─────────────────┘   └───────────┘   └─────────────────┴───────┘
        TRRS ──────────────┘     └────────────── TRRS
                              USB-C
                                │
                               PC
```

- **左手**: 6列×4行 + サムクラスタ（MX, 3DP Hotswap, Curvatureあり, Tenting 12°）
- **中央**: 3列×4行 テンキー（MX, フラット, USB-Cマスター）
- **右手**: 6列×4行 + サムクラスタ + 55mmトラックボール + Choc 6キー（MX+Choc混合）

## 3. 右手ユニット断面図（側面）

**これがOrthBallの核心。全エージェントはこの図を基準に設計せよ。**

```
                                         手の位置
                                    ┌──────────────┐
                        指先 ───▶   │  ╲  ╲  ╲  ╲  │   ◀─── 指先（ボール操作）
                                    └──┬───────┬───┘
  ───── キートップ面 ─────────────────  │       │
  ~26mm                               │       │       ┌───────────┐
  ─────┬─────────────────────────────  │       │    ↗  │           │  ~35mm
       │ ケース上面                     │       │  ／   │   ◉ ボール │  ボール中心
       │                               │       │／     │   55mm    │
  ~20mm│ MXスイッチ+PCB               │       ├───── │           │
       │ ┌─┐ ┌─┐ ┌─┐ ┌─┐           │       │      └─────┬─────┘
       │ │S│ │S│ │S│ │S│  PCB ──────┤  壁   │   BTU×3   │
       │ └─┘ └─┘ └─┘ └─┘           │       │   ベアリング │
  ─────┴───────────────────────────── ┴───────┤      PMW3360
       │ ケース底面 + ゴム足 (~3mm)            │           │
  ═════╧═══════════════════════════════════════╧═══════════╧══ 机面

  ←────── キーエリア ──────→ ←─壁─→ ←─ ボールエリア ─→
```

### 寸法の目安（推定値 — 要Shudai実測で確定）

| パラメータ | 目安値 | 根拠 |
|-----------|--------|------|
| **机面〜キートップ** | ~26mm | MXスイッチ(~16mm) + PCB(1.6mm) + ケース底(3mm) + キャップ(MT3 ~5mm) |
| **机面〜ボール中心** | ~35mm | On the 17参考。ボール半径27.5mm、下半分保持で中心は底面+35mm付近 |
| **ボール中心とキー面の高低差** | +9mm（ボールが上） | 指先で上面操作するにはキー面より少し高い必要がある |
| **ボール露出（上面）** | ~40% | 上から見て指先が触れる面積 |
| **ボール露出（側面）** | ~25% | 親指/手のひらで横から操作 |
| **ボール露出率合計** | 60-70% | 上面+側面の合計 |
| **ケース全体高さ（最高点）** | ~40mm | ボールカップ上縁付近 |
| **ケース幅（右手ユニット）** | ~160mm | キーエリア~120mm + ボールエリア~40mm |
| **Tenting角度** | 12° | Cosmos設定値 |

### ⚠️ Shudaiへのお願い

**On the 17を実測して以下を教えてほしい:**
1. 机面〜キートップの高さ
2. 机面〜トラックボール中心の高さ
3. ボールの露出具合（上から何%見えるか）
4. 操作時の手首の角度感

→ 実測値が入ったらこのドキュメントの寸法を確定する

## 4. 操作イメージ

### 基本姿勢
```
                    ┌── 人差し指〜小指: キーホームポジション
                    │
      ╲  ╲  ╲  ╲   │   ╱ ─── 親指: サムクラスタ or ボール側面
       ▼  ▼  ▼  ▼  │  ▼
  ┌──────────────┬──┴──┬────────┐
  │  キーエリア   │  壁  │  ◉ TB  │
  └──────────────┴─────┴────────┘
```

### 操作モード

1. **タイピング時**: 通常のホームポジション。トラックボールには触れない
2. **カーソル操作**: 右手を少し右にスライド → 人差し指/中指でボール上面を操作
3. **精密操作**: 親指でボール側面を操作（手のひらの付け根がケースに載る）
4. **クリック操作**: ボール下のChocキーで左クリック/右クリック/スクロール/中クリック
5. **ブラウザ操作**: ボール上のChocキーで戻る/進む

### ポイント
- **手を大きく動かさない**: ホームポジションからの移動距離は3-4cm程度
- **上面操作 = 大きなカーソル移動**: 指先でぐりぐり
- **側面操作 = 細かい操作**: 親指で精密コントロール
- **Expert Mouse的な操作感**: 大玉を手のひら全体で包む感じ

## 5. ボールカップ詳細

```
     上面開口（指先アクセス）
         ┌─────┐
        ╱       ╲
  側面 │    ◉    │ 側面
  開口 │  55mm   │ 閉鎖
       │ ボール  │（ケース壁）
        ╲  ▼▼▼  ╱
         └──┬──┘
            │
     BTU×3 (120°等配、下側寄り)
            │
        PMW3360（真下、上向き読み取り）
```

- **カップ形状**: 浅いお椀型。下半分（180°以下）のみ保持
- **側面開口**: キーエリアと反対側（右外側）には壁。キーエリア側に開口
  - → 実際にはキーエリアと**同じ側**に開口する方が親指でアクセスしやすい？
  - → **⚠️ 要議論: 開口方向はShudaiと決定する**
- **BTUベアリング**: 3個、120°等配、ボール赤道より下に集中
- **センサー**: PMW3360、ボール直下に上向きマウント

## 7. 各エージェントへの指示

### 🎨 CAD Designer

**あなたが作るもの: 3Dプリント可能なケース**

- 右手ユニットのケースが最重要・最難関
- ボールカップは「浅いお椀」— 下半分保持、上面+側面に開口
- **セクション3の断面図の寸法を厳守**（確定後）
- ボール中心はキー面より+9mm高い位置
- 壁の厚さ: 2-3mm（3Dプリント最小壁厚を考慮）
- BTUベアリングのポケット（3箇所）を正確に配置
- PMW3360のマウント穴をボール直下に
- Choc 6キーのマウント位置をボールカップ周辺に
- Tenting角度12°をケース底面の傾斜で実現
- ゴム足の取付位置を底面に

### ⚡ PCB Engineer

**あなたが作るもの: 各ユニットのPCB**

- 右手PCBはMXキー + PMW3360 + Chocキーを1枚に統合
- **PCBからボールカップ底面までの高さを確認** → ケースの内部空間に収まるか
- PMW3360はPCB裏面マウント or 別基板？→ 高さ制約から決定
- TRRS接続: 中央テンキーとの通信
- MCU: 中央がマスター（USB-C）、左右はスレーブ
- BTUベアリングはPCBマウントではなくケースマウント

### 🔧 FW/HW Engineer

**あなたが作るもの: ファームウェアと部品選定**

- PMW3360 + 55mmボールの組み合わせ動作確認
- BTUベアリング: [仕様TBD] — 55mmボール対応サイズを選定
- Chocスイッチ: Kailh Choc v1（薄型、ボールカップ周辺に配置するため）
- QMK/Vial対応
- トラックボールのDPI設定、角度補正
- レイヤー切替でトラックボールの機能変更（カーソル/スクロール）
