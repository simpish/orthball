### トラックボールの操作方向

**上面+側面の両方から操作可能** ← 確定

```
側面図（右側）:
                    ↑ 上から指先で操作
         ┌──────┐
  キー行  │      │  ◐ ← ボール露出率60-70%
         │      │ /  \
         │  PCB ││cup │← 側面からも触れる
         │      │ \  /     （親指/手のひら）
         └──────┘────
```

- ボール露出率: **60-70%**（上面+側面）
- カップ形状: 浅いお椀型（下半分のみ保持）
- BTUベアリング: 下半分に集中配置（120°等配、下側寄り）
- PMW3360: ボール真下に配置（上向き読み取り）
- ケース側面: 開口部あり（ボールに横から触れるように）
- イメージ: ケンジントン Expert Mouse に近い
