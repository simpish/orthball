# TrackBall Cup Design

*Created: 2026-03-15*
*Status: v2 - BTUポケット付き、プリント検証中*

## コンセプト

55mmトラックボール用のお椀型カップ。独立モジュールとして設計し、ケース本体に設置（置く）方式。On the 17のようなむき出し感を目指す。

## 形状パラメータ

| パラメータ | 値 | 備考 |
|-----------|-----|------|
| 内径 | 57mm (半径28.5mm) | BTU突出1.1mmを考慮し55mmボールより大きく |
| 外径 | 67mm (半径33.5mm) | |
| 壁厚 | 5mm | 外側球→Solidify内向きで生成 |
| 斜めカット角度 | 10° | X軸周りに傾斜。奥壁が高く、手前が低い |
| カットZ位置 | 0.8mm | ボール中心よりわずかに上 |
| 底面カットZ | -20mm | センサーアクセス用の大開口 |
| 底面 | オープン | センサー(PMW3360)用 |

## BTU配置 (VCN311-7.5)

### BTU寸法

| 寸法 | 値 |
|------|-----|
| D (本体外径) | 9mm |
| D1 (圧入穴径) | 7.5mm (+0.04/+0.01) |
| L (圧入部深さ) | 4mm |
| H (フランジ高さ) | 1mm |
| L1 (ボール突出) | 1.1mm |
| d (メインボール径) | 4mm (POM) |
| 許容荷重 | 6N (VCN311) |

### ポケット形状（段付き穴）

```
内壁面
  │← 9.2mm →│
  ┌──────────┐  1mm (フランジ座)
  └──┬────┬──┘
     │7.6mm│     4mm (圧入穴)
     └────┘
     → 壁の中 →
```

- フランジ座: Φ9.2mm x 1mm深 (9mm + 0.2mm 3Dプリント公差)
- 圧入穴: Φ7.6mm x 4mm深 (7.5mm + 0.1mm 3Dプリント公差)
- 合計深さ: 5mm (壁厚と同じ → 一部貫通あり)

### 配置位置

- **3個、120°等配**
- 緯度: -25° (赤道から25°下)
- 配置半径: 28mm (内壁57mmより1mm内側 = 56mm径の位置)
- 方位角: 90° (奥), 210° (手前左), 330° (手前右)

```
上面図:
        BTU_1 (90°, 奥)
           ●
     ╭─────┤─────╮
    │             │
    │    ◉ Ball   │   ← キー側 (-Y, 壁が低い)
    │             │
    ╰──●─────●──╯
  BTU_2      BTU_3
  (210°)     (330°)
```

## Blender生成方法

1. 外側球 (半径33.5mm, segments=96, ring_count=48)
2. Bisect: 10°傾斜カット + 底面カット (Z=-20mm)
3. Solidify: thickness=5mm, offset=-1 (内向き), use_rim=True
4. BTUフランジ3個を結合 → Boolean EXACT (Difference)
5. BTUボディ3個を結合 → Boolean EXACT (Difference)

**重要**: フランジとボディは別々にBooleanすること。結合してからだと内部共有面でBoolean失敗する。

## 検証ポイント

- [ ] BTU VCN311がポケットにはまるか
- [ ] 55mmボールがBTU上でスムーズに転がるか
- [ ] ボールの安定性（横に押して外れないか）
- [ ] 底面開口からPMW3360が適切な距離(2.4mm)に配置できるか
- [ ] 壁厚が3Dプリントに十分か
